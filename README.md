# Futuristic Login Interface 🔐

A sleek, futuristic login screen built with HTML, CSS, and JavaScript.  
It features animated grid lines, floating neon shapes, and smooth UI interactions for a sci‑fi inspired authentication experience.

## ✨ Features

- Animated background grid created dynamically with JavaScript  
- Floating geometric shapes with randomized size, position, and animation timing  
- Modern login card with email and password fields  
- Password visibility toggle (eye / eye‑slash icon)  
- Fake authentication flow with status text updates:
  - "Authenticating..."
  - "Access Granted"
  - "System Ready"  
- Clean, responsive layout suitable for desktop and laptop screens

## 🧱 Tech Stack

- **HTML5** – structure for the login form and layout  
- **CSS3** – futuristic styling, gradients, and animations  
- **JavaScript (Vanilla)** – background grid/shape generation, password toggle, and login status handling
